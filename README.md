## Spielcharakter

C#-Projekt zur Modellierung eines Spielcharakters mit einer einfachen MVC-Struktur.


## Struktur

- `Model/` – Daten/Logik des Spielcharakters (z. B. Name, Level, Attribute)
- `View/` – Darstellung des Spielcharakters (z. B. Konsolenausgabe)
- `Control/` – Steuerung, verbindet Model und View
- `Program.cs` – Einstiegspunkt der Anwendung
- `Spielcharakter.csproj` / `.sln` – Projekt-/Solution-Dateien


## Lernziele

- Verständnis für:
  - Objektorientierung in C#
  - einfache Aufteilung in Model / View / Control
  - Projektstruktur in Visual Studio
- Basisidee: die Logik vom „Wie es angezeigt wird“ trennen.
